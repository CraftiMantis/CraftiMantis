### 👋 About

Embedded & robotics systems — **Jetson, CUDA, ROS 2, motor control, board bring-up**.
I publish the hard-won, *undocumented* bring-up work that otherwise leaves people stuck
on dead-end forum threads. Rigor over hype: every repo is reproducible and honest about
what works and what doesn't.

### 🔭 Currently working on

- 🔬 Bringing **EOL Jetson Xavier** back onto a modern stack (JetPack 6 — keeping the GPU alive)
- 📡 **Jetson Orin NX** Intel / MediaTek Wi-Fi drivers on JetPack 7.2 &nbsp;·&nbsp; *published*
- 🔌 **Seeed A203-V2 + Xavier NX** flash / device-tree / UART / CAN bring-up &nbsp;·&nbsp; *published*
- ⚙️ **CubeMARS AK80-9 V3** motor + `ros2_control` drivers over SocketCAN
